# JavaScript Notes – Structured Chaos

This repo is a collection of my JavaScript learning — broken into bite-sized, logically ordered chunks. It's not a course. It's not for teaching. It's **just clean, commented code** to revisit concepts without rewatching hours of videos.

## 📂 Folder Structure

Each folder = one step in the learning journey.

| Folder              | What’s inside                                 |
|---------------------|-----------------------------------------------|
| `01_basics`         | Variables, data types, math, date             |
| `02_basics`         | Arrays, objects, JSON                         |
| `03_basics`         | Functions, scopes, execution context          |
| `04_control_flow`   | if, else, ternary, truthy/falsy               |
| `05_iterations`     | for, while, map, filter, etc.                 |
| `06_dom`            | DOM selection, traversal, manipulation        |
| `07_projects`       | Mini DOM projects                             |
| `08_events`         | Events, bubbling, delegation                  |
| `09_advance_one`    | Promises, async/await                         |
| `10_classes_and_oop`| Prototypes, classes, inheritance              |
| `11_fun_with_js`    | Closures, call/apply/bind, weird edge cases   |

## 🚀 How to Use

You don’t need to “run” anything. Just open files in your browser.  
Skim the JS. Understand the logic. Close tab. Done.

No bundlers. No compilers. No `npm install`.  
Just raw, vanilla JS — like it’s 2005 but with modern syntax.

## 👀 Who’s This For?

- Me, mainly.
- You, if you’re sick of 40-minute tutorials for 5 lines of code.
- Anyone who wants to see real examples, not “todo apps.”
